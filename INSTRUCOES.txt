🤖 AUTOMATIZADOR ATIVIDADES v3.2
=================================

✅ COMO USAR (2 MINUTOS/SEMANA):

1. DESCOMPACTE este ZIP

2. BAIXE da plataforma Educação Profissional (não precisa ser programador):
   - Entre em Educação Profissional
   - Vá para a disciplina que deseja avaliar (ex: 2DS_Logic, 2DS_Redes, etc.)
   - Clique em Relatórios
   - Selecione Conclusão de Atividades
   - No final da página, use o link "Download em formato compatível com Excel (.csv)"

3. GUARDE os arquivos .csv no formato padrão:
   - TURMA_DISCIPLINA.csv
   - Ex: 2DS_Logica.csv, 2DS_Redes.csv, 2DS_Processos.csv, 2DS_Carreira.csv
   - O nome correto é fundamental para o programa reconhecer cada disciplina

4. COLE os CSVs na pasta csv_brutos/

5. DUPLO CLIQUE em Automatizador_Atividades.exe

5. SELECIONE turma + digite SEMANA Atual que queira verificar (ex: 8)

6. CLIQUE 🚀 EXECUTAR

7. AGUARDE 30-60s → VER RESULTADO:
   📁 painel_turmas/ ← Excel turmas
   📁 pendencias_detalhadas/ ← 3 relatórios:
      - pendencias_detalhadas_<turma>.csv
      - controle_alunos_<turma>.csv (Q1,Q2,Q3,R por semana)
      - controle_alunos_resumo_<turma>.csv (totais de registros/quizzes)
   ✉️ avisos_alunos/ ← Planilha com mensagem para cada aluno das pendências
   📋 debug.log ← detalhes

=================================
🔧 PERSONALIZAR (Bloco de Notas):

config.ini:
[TURMAS]
lista = 2DS,3DS,2ADM,3ADM  ← suas turmas

[SEMANAS] 
padrao = 7           ← semana inicial

Salve → reinicie programa!

=================================
❓ PROBLEMAS?
- Ctrl+C no LOG → copia erro
- Abra debug.log no Bloco de Notas
- Envie para suporte

✅ Funciona QUALQUER Windows
